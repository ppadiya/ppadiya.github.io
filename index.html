Featured Projects

Credit Card Rewards Program

A recurring friction point in loyalty programs is the gap between what a bank promises and what a cardholder actually experiences: cluttered dashboards, opaque point balances, and redemption flows that lose users mid-process.

This project is a fully functional UI prototype for a credit card rewards platform, built to demonstrate what a well-designed member experience looks like. It covers point balance display, transaction history, and a guided redemption flow, all responsive across mobile and desktop. Informed by the Security Bank BetterBanking Rewards Program, which won the Retail Banker International Asia Trailblazer Award 2025.

Tech: HTML5, CSS3, JavaScript

APAC Market Expansion

Mastercard's payment card benefit products were built for Western markets: single currency, English-only, and incompatible with regional compliance frameworks across Southeast Asia. As Product Manager for APAC, I led the localization effort across eight markets, introducing multi-currency pricing models, multi-language UX, and country-level regulatory compliance for markets including the Philippines, Thailand, and Singapore.

The outcome was a portfolio of locally compliant, market-ready products that generated $10M+ in new revenue. Delivery required cross-functional coordination across legal, compliance, engineering, and local business development teams, run end-to-end from discovery through commercial launch.

Tech: Market Analysis Tools, Localization Frameworks, CRM Systems

API Ecosystem Development

Mastercard's payment card benefits were delivered through a fragmented set of point-to-point integrations: each issuer required custom connectivity, stretching onboarding timelines to months and inflating per-partner maintenance costs. I led the design and delivery of a unified API gateway that standardised how issuers connect to the benefits layer.

The gateway introduced a consistent REST interface and shared Swagger documentation, halving the technical lift for new integrations. Third-party integration costs fell by 30% within the first year. New partner onboarding dropped from months to weeks, a 50% reduction in time-to-live that directly unlocked pipeline that was previously stalled on technical readiness.

Tech: RESTful APIs, API Gateway, Swagger/OpenAPI
